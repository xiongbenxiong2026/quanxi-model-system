# 全犀模型系统

全犀私域直播平台的数据展示与业务管理系统，为 **直播间** 和 **品牌方** 提供透明的数据服务。

## 系统结构

```
全犀模型系统/
├── index.html          # 首页/角色选择（深色主题 Landing Page）
├── studio.html         # 直播间后台 - 查看入驻品牌、竞对情报、流量方资源
├── brand.html          # 品牌方后台 - 查看直播间资源、人群画像、订单数据
├── admin.html          # 管理后台 - 增删改查所有数据
├── css/
│   └── style.css       # 全局样式
├── js/
│   ├── data.js         # 数据层 - 模拟数据生成 + localStorage CRUD
│   ├── admin.js        # 管理后台逻辑
│   ├── studio.js       # 直播间后台逻辑
│   └── brand.js        # 品牌方后台逻辑（含 Chart.js 图表）
└── README.md
```

## 功能说明

### 直播间后台
- 品牌入驻看板：查看平台上所有入驻品牌（脱敏）、品类、产品信息
- 竞对情报：某某甄选、某某优选等行业头部平台数据对比
- 流量方资源：线下拉新服务商 + 打粉团队一览

### 品牌方后台
- 直播间资源：50+合作直播间概览（脱敏），流量类型、受众规模
- 人群画像：年龄/性别/消费能力/品类偏好/流量来源/复购率 6 大图表
- Top 10 品牌案例：销量排行榜，带进度条
- 订单数据统计：近30天交易额、订单趋势图（双轴）

### 管理后台
- 直播间管理：增删改查 50 个模拟直播间
- 品牌方管理：增删改查 300 个模拟品牌方
- 订单管理：1500+ 条订单记录
- 流量方管理：12 个合作流量渠道

## 运行方式

直接用浏览器打开 `index.html` 即可，无需服务器。

数据存储在浏览器 localStorage，首次打开会自动初始化模拟数据。

## 数据说明

- 所有品牌名和直播间名均已脱敏（使用「***」代替真实名称）
- 数据纯属演示用途，不涉及真实业务数据
- 内部统一称为「商务数据」

## 技术栈

- 纯前端 HTML5 + CSS3 + JavaScript
- Chart.js 4.x 数据可视化
- localStorage 数据持久化
- 无外部依赖（Chart.js 使用 CDN）
